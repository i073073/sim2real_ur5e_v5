# UR5e 제어 모델 개선 전략 및 구현 보고서

## 1. 모델 개선 전략: RL > GA > Bayesian Optimization

**개요:** 이전 단계에서 모방 학습(Imitation Learning)을 통해 전문가 궤적을 모방하는 초기 제어 모델을 확보했습니다. 이 모델은 정적인 환경에서 전문가의 행동을 재현하는 데는 효과적이지만, 실제 환경의 불확실성(Sim2Real Gap)에 적응하거나 최적의 성능(예: 최소 시간, 최소 에너지)을 달성하도록 최적화하는 데는 한계가 있습니다.

본 개선 전략은 AI/ML 전문가로서의 우선순위(강화 학습 > 유전 알고리즘 > 베이지안 최적화)에 따라, 모방 학습 모델을 기반으로 강화 학습을 수행하여 적응성을 높이고, 이후 최적화 기법을 적용하여 성능을 극대화하는 다단계 접근 방식을 채택합니다.

### 1.1. Phase 1: 강화 학습 (Reinforcement Learning) 기반 정책 개선 (Priority 1)

*   **목표:** 모방 학습 모델을 초기 정책으로 사용하여 시뮬레이션 환경에서 강화 학습을 수행합니다. 이를 통해 Sim2Real Gap에 강건하고, 정의된 보상 함수에 따라 최적화된 정책을 학습합니다.
*   **전략:** PPO(Proximal Policy Optimization) 알고리즘을 사용하여 정책을 업데이트합니다. PPO는 샘플 효율성이 높고 안정적인 학습이 가능하여 로봇 제어 분야에서 널리 사용됩니다.
*   **보상 함수 설계:**
    *   **목표 달성 보상:** 목표 위치에 도달하면 양의 보상.
    *   **거리 기반 보상:** 목표 위치와의 거리에 비례하여 음의 보상 (거리 감소 시 양의 보상).
    *   **제어 노력 페널티:** 관절 속도 및 가속도에 비례하여 음의 보상 (에너지 효율성 고려).
    *   **안전 페널티:** 충돌 발생 시 큰 음의 보상.

### 1.2. Phase 2: 유전 알고리즘 (Genetic Algorithm) 기반 하이퍼파라미터 최적화 (Priority 2)

*   **목표:** 강화 학습 정책의 성능을 극대화하기 위해 PPO 알고리즘의 하이퍼파라미터(예: 학습률, 배치 크기, GAE 람다)를 최적화합니다.
*   **전략:** 유전 알고리즘은 복잡한 탐색 공간에서 전역 최적해를 찾는 데 효과적입니다. RL 학습의 하이퍼파라미터는 서로 복잡하게 얽혀 있어 수동 튜닝이 어렵습니다. GA를 사용하여 다양한 하이퍼파라미터 조합을 탐색하고, 가장 높은 보상을 달성하는 조합을 찾습니다.

### 1.3. Phase 3: 베이지안 최적화 (Bayesian Optimization) 기반 실환경 미세 조정 (Priority 3)

*   **목표:** 시뮬레이션에서 학습된 정책을 실제 로봇에 적용하기 전, 실환경에서 발생하는 Sim2Real Gap을 최소화하기 위해 정책의 핵심 파라미터(예: 궤적 속도 계수, 안전 마진)를 미세 조정합니다.
*   **전략:** 베이지안 최적화는 적은 수의 실험으로 최적해를 찾는 데 유리합니다. 실제 로봇 실험은 시간과 비용이 많이 들기 때문에, BO를 사용하여 실환경에서의 성능을 평가하고 최적의 파라미터를 효율적으로 탐색합니다.
