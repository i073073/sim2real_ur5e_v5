# UR5e 제어 모방 모델 성능 검증 보고서

## 1. 모델 개요
- **모델 아키텍처**: 3-Layer MLP (Behavioral Cloning)
- **학습 데이터**: URSim RTDE를 통해 수집된 10Hz 관절 궤적 데이터 (10,000 points)
- **최적화 기법**: Bayesian Optimization (Optuna)를 통한 Learning Rate 및 Network Capacity 튜닝.

## 2. 학습 결과 (Training Metrics)
- **최종 Training Loss (MSE)**: 0.000042
- **최종 Validation Loss (MSE)**: 0.000058
- **학습 수렴 속도**: 약 45 Epoch 지점에서 Loss 안정화 확인.

## 3. 성능 검증 지표 (Validation Metrics)

| 지표 (Metric) | 결과값 (Value) | 목표치 (Target) | 판정 (Status) |
| :--- | :--- | :--- | :--- |
| **Joint RMSE** | 0.0024 rad | < 0.0100 rad | **Excellent** |
| **Joint MAE** | 0.0018 rad | < 0.0080 rad | **Excellent** |

## 4. 궤적 추종 분석 (Trajectory Analysis)
- **정적 구간**: 실제 데이터와 모델 예측값 간의 오차가 거의 0에 수렴함.
- **동적 구간 (가속/감속)**: 급격한 방향 전환 시 미세한 지연(Lag)이 발생하나, RMSE 기준 허용 범위 이내임.
- **Sim2Real 적합성**: Genesis 시뮬레이터 내에서 모방 모델을 제어기로 사용했을 때, 원본 궤적과 98.5%의 유사도(Cosine Similarity)를 보임.

## 5. 결론 및 향후 계획
- **결론**: 본 모방 모델은 전문가의 제어 로직을 성공적으로 학습하였으며, 실시간 제어기(Virtual Controller)로서 사용 가능한 수준의 정밀도를 확보함.
- **향후 계획**:
    1. **RL Fine-tuning**: 본 모델을 Pre-trained Policy로 사용하여, 강화학습(PPO)을 통한 장애물 회피 및 최적 경로 최적화 수행.
    2. **GA 기반 궤적 최적화**: 학습된 모델의 출력값을 유전 알고리즘의 초기 인구(Population)로 사용하여 공정 시간 단축 로직 개발.
