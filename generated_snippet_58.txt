
#### **4. 검증 결과**

##### **4.1. Genesis 시뮬레이터 시각화 정확성**

| 검증 항목 | 결과 | 판정 |
| :--- | :--- | :--- |
| **1. URDF 모델 로딩** | URDF 모델이 정상적으로 로드되어 UR5e 로봇의 형상이 정확하게 렌더링됨. 각 관절의 연결 부위 및 링크의 비율이 올바르게 표현됨. | **Pass** |
| **2. 관절 제어 정확성** | `[0, -90, 90, -90, -90, 0]` (deg) 자세 명령 시, 로봇이 팔을 수직으로 세우고 손목을 아래로 꺾는 예상된 자세를 정확히 취함. | **Pass** |
| **3. TCP 마커 정확성** | 로봇의 `tool0` 링크를 움직일 때마다 빨간색 구형 마커가 동일한 상대 위치(끝단에서 Z축으로 50mm)를 유지하며 함께 이동함을 확인. | **Pass** |
| **4. 궤적 트레일 정확성** | 로봇이 원을 그리는 동작을 수행할 때, 생성된 청록색 트레일이 TCP 마커의 중심점을 정확히 따라가며 부드러운 원형 경로를 형성함. | **Pass** |

**시각 자료:**
-   `reports/screenshots/genesis_pose_check.png` (관절 제어 자세 스크린샷)
-   `reports/screenshots/genesis_trajectory_check.png` (궤적 트레일 스크린샷)
