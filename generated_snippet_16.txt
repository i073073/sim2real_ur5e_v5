# UR5e 명령어 라이브러리 및 RTDE 통신 기능 검증 테스트 스크립트 및 보고서

## 1. 테스트 계획 및 목표

**테스트 목표:** UR5e 로봇 제어 시스템의 `Execution Layer` 핵심 구성 요소인 RTDE 통신 모듈 (`rtde_client.py`)과 명령어 라이브러리 (`ur5e_commands.py`)의 기능적 정확성, 신뢰성 및 안전성을 검증합니다.

**테스트 환경:**
*   **하드웨어:** UR5e 로봇 (또는 URSim 시뮬레이터)
*   **소프트웨어:** Python 3.10+, `rtde-python` 라이브러리, `pytest` 프레임워크
*   **테스트 대상 모듈:** `src/rtde_client.py`, `src/ur5e_commands.py`

**테스트 항목:**
1.  **RTDE 통신 연결 및 해제:** 로봇 컨트롤러와의 연결 설정 및 해제 기능 검증.
2.  **실시간 데이터 수신 및 로깅:** RTDE Output 데이터 수신 및 10Hz 로깅 기능 검증.
3.  **명령어 실행 기능 (`moveJ`, `moveL`, `speedJ`):** 각 명령어의 정상 작동 여부 및 로봇 동작 검증.
4.  **안전성 검증 (Joint Limit Check):** `ur5e_commands.py`에 구현된 관절 한계값 검사 기능 검증.

## 2. 테스트 스크립트 (`tests/test_ur5e_control.py`)

다음은 `pytest` 프레임워크를 사용하여 UR5e 제어 모듈을 검증하는 테스트 스크립트입니다. 이 스크립트는 URSim 시뮬레이터가 실행 중인 환경에서 실행되도록 설계되었습니다.
