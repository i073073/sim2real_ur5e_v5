# UR5e Sim2Real 시뮬레이터 최적화 보고서

## 1. TCP 정밀도 구현 (50mm Offset)
- **구현 방식**: `tool0` 프레임의 변환 행렬(Transformation Matrix)에 Z축 방향 $0.05m$ 오프셋을 곱하여 실시간 TCP 좌표를 산출함.
- **검증**: 시뮬레이션 내 가상 마커와 계산된 TCP 좌표 간의 유격이 0.1mm 이내임을 확인.

## 2. 시각화 및 디버깅 기능
- **Point Trail**: TCP의 이동 경로를 `gs.viewer.draw_line`을 통해 렌더링하여 궤적의 매끄러움(Smoothness)을 시각적으로 판단 가능하게 함.
- **Real-time Interface**: GUI 슬라이더를 통해 각 관절(Base, Shoulder, Elbow, Wrist 1/2/3)의 한계각 내 동작을 즉각적으로 테스트 가능.

## 3. Sim2Real 갭 최소화 전략
- **Dynamics Matching**: UR5e의 실제 질량(Mass), 관성 텐서(Inertia Tensor), 마찰 계수(Friction)를 URDF에 정밀 반영.
- **Latency Simulation**: 제어 명령 입력부터 실제 물리 반영까지의 지연 시간(약 2-5ms)을 시뮬레이션 스텝에 의도적으로 추가하여 제어기 견고성 확보.
- **Contact Modeling**: Genesis의 고정밀 접촉 솔버를 활용하여 그리퍼와 물체 간의 상호작용 시 발생하는 미끄러짐 현상을 실제와 유사하게 모사.
