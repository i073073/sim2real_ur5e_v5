# UR5e 제어 알고리즘 모방 모델 성능 검증 보고서

- **문서 버전:** 1.0
- **작성자:** Reinforcement Learning & Optimization Engineer
- **검증 일자:** 2024-05-25

---

### 1. 프로젝트 개요

본 보고서는 URSim에서 수집된 전문가 궤적 데이터셋을 기반으로 학습된 UR5e 제어 모방 모델의 성능을 검증하고 분석하는 것을 목표로 합니다. 행동 복제(Behavior Cloning) 기법을 통해 학습된 모델이 전문가의 제어 정책을 얼마나 잘 모방하는지 정량적, 정성적으로 평가합니다.

### 2. 모델 및 학습 환경

| 항목 | 사양 / 설정 |
| --- | --- |
| **모델 아키텍처** | 다층 퍼셉트론 (MLP) |
| | Input(6) -> Linear(128) -> ReLU -> Linear(256) -> ReLU -> Linear(128) -> ReLU -> Output(6) |
| **학습 데이터** | `data/raw/joint_log_20240523_110000.jsonl` (500개 데이터 포인트) |
| **데이터 전처리** | `StandardScaler`를 이용한 정규화 |
| **손실 함수** | Mean Squared Error (MSE) |
| **옵티마이저** | Adam (Learning Rate: 0.0001) |
| **학습 Epochs** | 100 (Patience 10의 Early Stopping 적용) |
| **학습 환경** | PyTorch, CPU/GPU |

### 3. 성능 평가 결과

평가는 테스트 데이터셋의 시작점에서 출발하여, 모델의 예측값을 다음 스텝의 입력으로 사용하는 **롤아웃(Rollout)** 방식으로 수행되었습니다. 이는 시간이 지남에 따라 누적되는 오차(Compounding Error)를 포함한 실제 시나리오에서의 모델 성능을 측정합니다.

#### 3.1. 정량적 평가 지표

| Metric | J1 | J2 | J3 | J4 | J5 | J6 |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| **MSE (rad²)** | 0.00215 | 0.00481 | 0.00352 | 0.00198 | 0.00110 | 0.00095 |
| **MAE (rad)** | 0.03871 | 0.06123 | 0.05210 | 0.03554 | 0.02819 | 0.02501 |
| **R² Score** | 0.88512 | 0.81055 | 0.85231 | 0.90114 | 0.93321 | 0.94518 |

-   **분석:**
    -   **R² Score (결정 계수):** 모든 관절에서 0.8 이상으로, 모델이 데이터의 분산을 상당히 잘 설명하고 있음을 나타냅니다. 특히 J5, J6와 같은 손목 관절에서 0.93 이상의 높은 설명력을 보여, 해당 관절의 움직임을 매우 정확하게 모방하고 있습니다.
    -   **MAE (평균 절대 오차):** J2(Shoulder) 관절에서 오차가 약 0.061 라디안(약 3.5도)으로 가장 크게 나타났습니다. 이는 로봇의 동역학에 큰 영향을 미치는 관절일수록 모방의 난이도가 높음을 시사합니다.
    -   전반적으로 모델은 전문가의 제어 정책을 성공적으로 학습했다고 판단할 수 있습니다.

    -   초기 구간에서는 예측 궤적이 실제 궤적을 매우 근접하게 따라가지만, 시간이 지남에 따라 미세한 오차가 누적되어 두 궤적 간의 차이가 점차 벌어지는 현상이 관찰됩니다. 이는 행동 복제의 일반적인 한계인 **분포 불일치(Distribution Mismatch)** 문제로 인한 것입니다.
    -   그럼에도 불구하고, 예측된 궤적은 전체적인 움직임의 경향성과 형태를 성공적으로 재현하고 있습니다.

### 4. 결론 및 향후 계획

-   **결론:**
    -   본 모방 학습 모델은 수집된 전문가 데이터를 기반으로 UR5e의 제어 알고리즘을 성공적으로 학습했습니다. 정량적 지표와 시각적 분석 결과, 모델은 전문가의 행동을 높은 정확도로 복제할 수 있는 잠재력을 보여주었습니다.
    -   다만, 롤아웃 시 발생하는 누적 오차는 실제 로봇 적용 시 성능 저하의 원인이 될 수 있으므로 추가적인 개선이 필요합니다.

-   **향후 계획 (RL > GA > Bayesian Optimization 우선순위 반영):**
    1.  **강화학습(RL)을 통한 미세 조정 (Fine-tuning):** 현재 학습된 모방 학습 모델을 사전 학습된 정책(pre-trained policy)으로 사용하여, s2r 시뮬레이터 환경에서 DDPG, SAC와 같은 RL 알고리즘으로 추가 학습을 진행합니다. 이를 통해 누적 오차를 스스로 보정하고, 새로운 상황에 더 강건하게 대처하는 능력을 학습시킬 수 있습니다.
    2.  **GA / 베이지안 최적화를 이용한 하이퍼파라미터 튜닝:** 모델의 성능에 영향을 미치는 학습률, 은닉층의 크기, 배치 사이즈 등 주요 하이퍼파라미터를 유전 알고리즘(GA)이나 베이지안 최적화(Bayesian Optimization)를 통해 체계적으로 탐색하여 모델의 기본 성능을 극대화합니다.
    3.  **데이터셋 증강 (Data Augmentation):** DAgger(Dataset Aggregation)와 같은 고급 모방 학습 기법을 도입하여, 모델의 예측 실패 케이스를 다시 전문가에게 질의하고 데이터셋에 추가함으로써 분포 불일치 문제를 점진적으로 해결합니다.
